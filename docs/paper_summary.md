# Paper summary

This repository supports the paper **Same Architecture, Different Capacity: Optimizer-Induced Spectral Scaling Laws** by Nandan Kumar Jha and Brandon Reagen.

- Paper: https://arxiv.org/abs/2605.21803
- Project website: https://optimizer-scaling-laws.github.io/
- Repository: https://github.com/optimizer-scaling-laws/spectral-scaling-laws

Central claim: models with the same architecture and similar pretraining loss can realize different representation capacity depending on optimizer geometry. The repo provides training code, spectral telemetry, token-frequency bucket analysis, and reproducibility scaffolding for the paper results.

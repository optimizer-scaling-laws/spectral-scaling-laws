# Sample logs

This directory contains small legacy submitted-run log snippets used for parser
examples and tests. They are intentionally not the full raw logs.

- `adamw_160m_1x_layer0_global_legacy.txt` — one global per-layer spectral telemetry file.
- `adamw_160m_1x_layer0_frequency_legacy.txt` — one HEAD/MID/TAIL frequency-bucketed layer telemetry file.

Full submitted-run logs should be distributed as external artifacts rather than
committed directly to the repository.
